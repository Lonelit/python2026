# команды GIT

## git status
### состояние репозитория

## git add file.py
### добавить в индекс файл file.py

## git add .
### добавить в индекс все файлы в текущем каталоге

## git commit -m "comment for commit"
### создать коммит с комментарием commet for commit

## git push
### залить изменения с локального репозитория github

## git pull 
### скачать изменения с github в локальный репозиторий

## git remote
### управление связью между текущим и удаленным репозиторием

## git log --oneline
### упрощенная история комитов
